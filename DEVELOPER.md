## Key Components
- **`app.py`**: Contains the main logic for the Streamlit app.
- **YFinance**: Fetches stock data and news.
- **Plotly**: Creates interactive charts.
- **Streamlit**: Provides the web interface.
- **Scikit-learn**: Provides ML algorithms.
